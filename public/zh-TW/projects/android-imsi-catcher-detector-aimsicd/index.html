Android IMSI-Catcher Detector (AIMSICD)
蜂巢式網路

Android open-source based project to detect and avoid fake base stations (IMSI-Catchers) or other base-stations (mobile antennas) with poor/no encryption. This project aims to warn users if the ciphering is turned off and also enables several other protection-mechanisms. Since it is under constant development, we constantly search for courageous testers and security-enthusiastic developers. Don't be shy, feel free to contribute, in any way you can. We're on GitHub and XDA. Thank you for participating in development! ;-)
