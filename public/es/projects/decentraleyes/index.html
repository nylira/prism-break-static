Complementos para navegadores Web

Protégete contra rastreo a través de entrega de contenido centralizado "gratis". Previene interferencia de redes como Google Hosted Libraries, sirve archivos locales para prevenir que sitios se rompan. Complementa bloqueadores regulares.

Notas de Seguridad
  • Protege la privacidad al evadir una larga red de entrega que dice ofrecer sus servicios gratuitamente.
  • Complementa las aplicaciones de bloqueo regulares tales como uBlock Origin, Request Policy, y otros.
  • Funciona directamente al ser instalado; Absolutamente ninguna configuración previamente requerida.

Decentraleyes no es una bala de plata, pero sí previene que muchos sitios web envíen este tipo de peticiones. Así también, Decentraleyes puede bloquear peticiones para cualquier recurso CDN que no se pueda encontrar.
