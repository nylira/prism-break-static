Notas

El robusto proyecto etherpad mantiene una lista de sitios que corren el servicio etherpad. Por favor escoge sólo entre aquellos que usan SSL, e investiga el contexto del sitio antes de confiarle tus datos.

Riseup también ofrece correo electrónico, XMPP, y mensajería instantánea, los cuales están accesibles a través de direcciones de servicios secretos de Tor. La lista de dichas direcciones está aquí. (inglés)

ProtectedText encrypts/decrypts text in the browser, and password (or it's hash) is never sent to the server - so that text can't be decrypted even if requested by authorities.
