Notes

Le stockage de fichiers en ligne est également disponible via les composants de Kolab Chwala/iRony avec la possibilité d'intégrer différents backends de stockage. Les fichiers sont accessibles via les options d'accès de couche de stockage, WebDAV et Kolab, des interfaces web intégrant Roundcube.

Tarsnap n'est pas recommandé sur PRISM Break en raison de son manque d'options de paiement anonymes et un copyright strict sur le client qui rend difficile le remplacement du service en cas d'arrêté.

BitTorrent Sync, MEGA, et SpiderOak sont des services qui sont construits sur partiellement ou totalement le logiciel propriétaire. Ils ne seront pas recommandé sur PRISM Break tant que la totalité de leur code ne sera pas open source.

Avec un logiciel à code source fermé, vous devez avoir une confiance totale du vendeur. Même si vous pouvez porter garant de leur intégrité, le logiciel propriétaire a toujours plus de bogues et failles de sécurité non interceptées parce qu'il y a moins de personne pouvant examiné le code source.

Une autre alternative pour le stockage en ligne serait une sauvegarde locale avec les disques durs externes et les lecteurs flash USB. Cette méthode est fiable et plus sécurisé que le stockage des données sur un réseau, mais a un coût.
