Open Source Routing Machine
Картографические сервисы

Реализация на C++ высокопроизводительной системы навигации для поиска кратчайших путей в дорожных сетях. Лицензирована под пермиссивной 2-пунктовой лицензией BSD. OSRM поддерживает Linux, FreeBSD, Windows и Mac.
