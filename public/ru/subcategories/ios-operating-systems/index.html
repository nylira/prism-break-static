Примечания

Apple, Google и Microsoft часть проекта PRISM. Их проприетарные операционные системы не могут быть доверены для защиты вашей персональной информации от АНБ.

У нас есть две свободные альтернативы: GNU/Linux и BSD.

GNU/Linux имеет намного большее сообщество, которое может помочь вам с переходом. Рекомендуется начать с одного из дистрибутива GNU/Linux, который подойдет под ваши нужны.

Debian имеет давние традиции в области свободного программного обеспечения. Разработчики должны подписать социальный контракт и придерживаться этического манифеста. Строгие критерии включения ПО позволяют убедиться, что только сертифицированное открытое программное обеспечение будет поставляться в главном репозитории.

Gentoo является мета-дистрибутивом. Исходный код компилируется в бинарные приложения на пользовательском компьютере, что позволяет использовать практически безграничные возможности для адаптации и полную обратную прослеживаемость программной логики.

И Fedora, и openSUSE являются версиями для сообщества, которые служат стабильной основой для промышленных дистрибутивов GNU/Linux c коммерческой поддержкой. Компании по всему миру верят Red Hat Inc. и SUSE Linux GmbH из-за прозрачности всего процесса разработки.

Parabola GNU/Linux фактически на 99% состоит из Arch Linux с ядром, из которого удалены блобы и мета-пакетом, который блокирует несвободные лицензии. Это же можно и проделать с Arch.

Ubuntu от Canonical не рекомендуется PRISM Break, потому что имеет рекламу от Amazon и утечки информации по умолчанию. GNU/Linux дистрибутивы, основанные на Ubuntu, также пока не рекомендуются из-за некоторых других причин.
