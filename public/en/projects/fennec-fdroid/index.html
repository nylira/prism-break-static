Security Notes

Fennec FDroid does not hide your IP address from third-parties by default. If you want to anonymize your IP, consider using Orbot.

Firefox sends searches through Google by default. Consider changing it to DuckDuckGo or Startpage for anonymous web searches.

Considering unchecking "Block reported attack sites" and "Block reported web forgeries" if you don't want Mozilla to send reported URL hashes to Google.
