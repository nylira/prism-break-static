Note di Sicurezza

Pidgin memorizza le password dei tuoi account di chat in chiaro. Se vuoi evitarlo puoi (1) non salvare le password in Pidgin, (2) criptare il tuo file system, oppure (3) salvare le tue password di Pidgin in modo sicuro con il pacchetto Debian pidgin-gnome-keyring.

Pidgin con OTR e dbus attivi ha un bug di sicurezza locale. Vedi la segnalazione del bug per maggiori informazioni e possibili soluzioni.
