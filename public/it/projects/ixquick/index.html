Note di sicurezza

Ixquick returns the top ten results from multiple search engines.

In contrast to ixquick.eu, its sibling project Startpage.com offer results from the Google search engine.

It also provides the stand-alone Ixquick Proxy service that allows users to open search results via proxy.
