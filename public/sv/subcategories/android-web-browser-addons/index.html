Skydda din integritet och hindra webbsidor från att spåra dig genom att installera uBlock Origin, Request Policy, HTTPS Everywhere, och NoScript i din webbläsare.

Ett mera avancerad alternativ till Request Policy skulle vara uMatrix.

Varför inte Adblock Plus? Adblock Plus visar “acceptabla annonser” som standard, och använder mer arbetsminne än uBlock Origin.
