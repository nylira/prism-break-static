2014-05-28: TrueCrypt は現在セキュリティ上の問題があり、推奨されていません。

GNU/Linux を実行している場合、 dm-crypt と LUKS は推奨される暗号化オプションです。

Gentoo GNU/Linux は dm-crypt と LUKS の ガイドをメンテナンスしています。

Arch GNU/Linux は dm-crypt と LUKS への ガイドをメンテナンスしています。 Parabola GNU/Linux でも使用できます。

Slackware Linux は dm-crypt と LUKS への ガイドをメンテナンスしています。

この文脈で 透過性 とは、(dm-crypt のような) 暗号化ツールによってすべてのプログラムに等しく提供されるため、個々のプログラムが独自のデータの暗号化を管理する必要はないことを意味します。 これに関する詳細は Wikipedia で。
