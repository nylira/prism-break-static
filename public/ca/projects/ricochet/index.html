Security Notes

Ricochet is a peer-to-peer instant messaging system built on Tor hidden services. It uses hidden service addresses as logins. Contacts connect to each other (not an intermediate server) through Tor, which is bundled with the IM client. The rendezvous system makes it extremely hard for anyone to learn a user's identity from the address.

Ricochet is not affiliated with or endorsed by The Tor Project.

Ricochet is a substitute for the obsolete TorChat, which is no longer maintained.
