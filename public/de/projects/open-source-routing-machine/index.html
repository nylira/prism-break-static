Open Source Routing Machine
Karten

Die Open Source Routing Machine (OSRM) ist eine C++ Implementierung eines hoch-performanten Streckenfindungsalgorithmus im Straßennetz. Lizensiert unter der großzügigen 2-Klausel BSD-Lizenz ist OSRM ein freier Netzwerkdienst, der Linux, FreeBSD, Windows und Mac unterstützt.

Sicherheitshinweise

OSRM sollte lokal installiert und betrieben werden, damit Datenlecks von unverschlüsselten Onlinediensten vermieden werden.
